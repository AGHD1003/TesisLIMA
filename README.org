* TeX Live 

Bajar el archivo ~texlive2013.iso~.

Correr

#+begin_src sh
sudo mount -t iso9660 -o ro,loop,noauto texlive2012.iso /mnt
#+end_src

luego ~cd /mnt~ y luego  ~sudo ./install-tl~
